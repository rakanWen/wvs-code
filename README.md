## 📊 Evaluation & Training

This repository primarily contains the **evaluation code** for our model. 

**Training Details:**
The model training, including both Supervised Fine-Tuning (SFT) and Direct Preference Optimization (DPO), was conducted using **[LLaMA-Factory](https://github.com/hiyouga/LLaMA-Factory)**. If you are interested in reproducing the training process or learning more about the training configurations, please refer to their official repository.
